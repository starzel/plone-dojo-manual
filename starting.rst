Running Plone
=============

Start Plone in *debug-mode*:

..  code-block:: bash

    $ ./bin/instance fg

Now when you open your browser on http://localhost:8080 you can see your site. You can stop Plone with ``ctrl+c``.

Run Plone in *production-mode*:

..  code-block:: bash

    $ ./bin/instance start

Stop Plone:

..  code-block:: bash

    $ ./bin/instance stop
